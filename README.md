## アプリケーション名
Sync Watch

## アプリケーション概要
離れた場所にいる人と同じ映画を同時に視聴しながらチャットで交流できるアプリ
視聴者数のカウント機能やお気に入り機能を搭載し映画を通じたコミュニケーションを楽しめる

## 利用方法
### 映画を視聴する
1. トップページからユーザー新規登録をする

2. ユーザー新規登録後、トップページの映画一覧から、視聴したい映画の詳細ページへ移動する。

3. 映画詳細ページで"視聴開始"ボタンを押すと、視聴者数がカウントされ、チャットページに参加できる。

4. "チャットに参加する"ボタンでチャットページに移動し、他の視聴者とリアルタイムで感想を共有できる。

5. 視聴を終了したい場合は"視聴終了"ボタンを押す。

### お気に入り機能
1.映画詳細ページの"お気に入り"ボタンを押すと、お気に入りリストに登録される。
2.ユーザー詳細ページで登録したお気に入り映画を確認できる。

## アプリケーションを作成した背景
映画は一人で観ても楽しいが、誰かと一緒に視聴して感想を共有することで、より深く楽しむことができる。
しかし、周囲に同じ映画を観る仲間がいないと、映画の感想や考察を話す機会が少なくなってしまう。

SNSで映画の考察を共有している人を見て、同じ作品をリアルタイムで視聴しながら語り合えたら、
映画体験がより豊かになるのではないかと考えた。

そこで、オンライン上で映画を同時視聴しているユーザー同士がリアルタイムで交流できるプラットフォームを作成し、
映画を観ながら自由に感想を語り合えるアプリ"Sync Watch"を開発した。

今後は映画だけでなく、ドラマやアニメの情報も取得し、視聴体験をさらに充実させていく予定。

## 実装予定の機能
今後実装予定の機能
- ドラマ・アニメの情報取得：TMDb APIを活用し、映画だけでなくドラマやアニメの情報も取得可能にする。
- 掲示板機能:ユーザーが自由に投稿でき、「何時からこの映画見よう」のような提案ができる。
- 自動視聴終了機能：一定時間視聴が続いた場合やユーザーがログアウトした場合は自動的に視聴を終了し、視聴者数を適切に管理する。
